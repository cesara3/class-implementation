# Freshman Engineering & ECE Prep Program

This project is a beginner-friendly engineering preparation course designed to help incoming students build confidence before taking courses such as Circuits I, Circuits II, Signals & Systems, Linear Algebra, and other introductory Electrical & Computer Engineering classes.

The purpose of the program is to bridge the gap between traditional math classes and real engineering coursework by focusing on intuition, visualization, and hands-on problem solving. Instead of immediately diving into heavy theory, the course emphasizes understanding concepts through examples, diagrams, and computational verification.

Students are introduced to engineering mathematics alongside beginner Python programming using Google Colab notebooks. Problems are first worked out by hand, then checked or visualized using Python. This helps students understand both the mathematics and how engineers use computation in practice.

---

# Topics Covered

- Complex numbers
- Powers of i
- Complex arithmetic
- Polar and rectangular form
- Euler’s formula
- Magnitude and phase
- Trigonometric functions
- Unit circle review
- Vectors and vector operations
- Dot products
- 2×2 matrices
- Determinants
- Matrix-vector multiplication
- Introductory probability and simulations
- Basic engineering problem solving
- Python fundamentals for engineering applications

---

# Learning Philosophy

This program follows a:
- Learn-by-doing approach
- Visualization-focused teaching style
- Hand-calculation-first mindset
- Engineering intuition development strategy

The goal is not to use Python as a shortcut, but as a tool for:
- verification,
- experimentation,
- visualization,
- and engineering exploration.

Students are encouraged to understand why methods work before relying on software tools.

# Goals of the Program

By the end of the course, students should:
- Feel more confident entering engineering coursework
- Understand foundational ECE mathematics
- Gain beginner programming experience
- Build stronger engineering intuition
- Become more comfortable learning technical material independently
- Understand how math, programming, and engineering connect together

# About

This project was created as part of an effort to improve early engineering preparation and provide incoming students with a smoother transition into Electrical & Computer Engineering coursework. The material is intentionally designed to be approachable while still maintaining real engineering relevance.
